1. O que é a Agenda Escolar

Visão geral

A Agenda Escolar 2026 é o calendário digital interativo da escola. Centraliza datas comemorativas, eventos pedagógicos, prazos institucionais, conselhos de classe, reuniões de pais e demais atividades do ano letivo.

PerfilO que pode fazer
Visitante (qualquer pessoa com o link)Consultar o calendário, ver eventos do mês, navegar entre meses
Administrador (senha)Adicionar, editar e remover eventos; exportar dados
🔑
A área administrativa é protegida por senha. Apenas a equipe diretiva e secretaria devem ter acesso.
🗓️

2. Navegando no Calendário

Consulta de eventos

1

Acesse a agenda

Link: emreginacelicr-tech.github.io/agenda-rc-2026/calendario_escola_regina_celi_2026.html

2

Navegue entre os meses

Use as setas ‹ e › nos cantos do calendário para avançar ou recuar entre os meses do ano.

3

Visualize eventos do dia

Clique em qualquer data marcada no calendário para ver os detalhes dos eventos daquele dia.

4

Filtrar por categoria

Use o menu de categorias para filtrar por tipo de evento: pedagógico, administrativo, comemorativo, etc.

⚙️

3. Área Administrativa

Adicionar e editar eventos

1

Acesse a área admin

Na agenda, procure o botão ou link "Admin" ou "Área Administrativa". Geralmente fica no rodapé ou em um ícone de engrenagem.

2

Insira a senha

Senha de acesso admin: direcao2026

3

Adicionar novo evento

Clique em + Novo Evento. Preencha: título, data, categoria, descrição e cor. Salve.

📌 Dica: use categorias consistentes para facilitar a filtragem posterior. Ex: "Pedagógico", "Administrativo", "Reunião de Pais".
4

Editar evento existente

Clique no evento no calendário → botão Editar. Altere os campos necessários e salve.

5

Remover evento

Abra o evento → botão Excluir. Uma confirmação será pedida antes de deletar permanentemente.

6

Sincronizar com GitHub

Após salvar eventos, use o botão Sincronizar / Publicar para que as alterações fiquem visíveis para todos. O sistema usa a API do GitHub para persistência dos dados.

💡 As alterações ficam salvas automaticamente no banco, mas é necessário publicar para que todos vejam.
📊

4. Exportando Dados

Relatórios e backup

AçãoComo fazer
Exportar para Excel/CSVNa área admin, procure o botão Exportar. Baixa todos os eventos em formato CSV compatível com Excel.
Imprimir o calendário do mêsNavegue até o mês desejado e use Ctrl+P (ou Cmd+P). A agenda tem layout otimizado para impressão.
Compartilhar linkCopie a URL e envie. Qualquer pessoa com o link pode consultar o calendário sem senha.
💡

5. Boas práticas de uso

SituaçãoRecomendação
Início do mêsRevise os eventos do mês e confirme se todas as datas estão corretas e atualizadas.
Mudança de data de eventoEdite imediatamente no sistema — não deixe para depois, para evitar confusão na equipe.
Eventos recorrentesCadastre um a um com datas específicas. Isso facilita edições individuais se a data de uma reunião mudar.
Datas comemorativas importantesCadastre com categoria "Comemorativo" e descrição com sugestão de atividade. Facilita o planejamento pedagógico.
Eventos canceladosEdite o evento e adicione "(Cancelado)" no título, ou remova se não houver necessidade de registro histórico.
